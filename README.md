FearLess
=======

Fearless is a Revolutionary Animation Production Management System

it's not ready yet,!


Requirements
===========
Fearless needs this requirements:

centos 6.6
python 2.7.8.10	
riak 2.0.4
mysql-server
jdk 7u25
ruby 2.0.0


Installaion
===========

Use the following commands:

```bash
sudo make prepare
make
sudo make install
```

Now you can visit [Home page](http://127.0.0.1/app)
